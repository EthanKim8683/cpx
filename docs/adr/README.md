## Format

- **Status** — Proposed, Accepted, Deprecated, or Superseded
- **Context** — What problem or question prompted the decision
- **Decision** — What we decided
- **Alternatives** — Other options discussed; write **None** if no alternatives were considered
- **Consequences** — Trade-offs, risks, and operational impacts; write **None** if none worth recording

# Architecture Decision Records

This directory records significant decisions about CPX.

| ADR | Title | Status |
|-----|-------|--------|
| [0001](0001-cpx-scope-and-boundaries.md) | CPX scope and boundaries | Accepted |

## Format

Each ADR follows a lightweight structure:

- **Status** — Proposed, Accepted, Deprecated, or Superseded
- **Context** — What problem or question prompted the decision
- **Decision** — What we decided
- **Consequences** — What follows from the decision

Product vision and ongoing discussion live in [CP-5](https://linear.app/ethankim8683/issue/CP-5/cpx-vision-goals-and-feature-discussion) / [GitHub #1](https://github.com/EthanKim8683/cpx/issues/1). ADRs record settled decisions; feature-specific technical ADRs will be added as implementation choices are made.
